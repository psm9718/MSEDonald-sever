## Requirements
For building and running the application you need:
- JDK 17
- SpringBoot 3.0.5
- Gradle 7.6.1

## Installing / Getting started

```
git clone https://github.com/psm9718/MSEDonald-sever.git
cd MSEDonald-server
./gradlw build
java -jar build/libs/*.jar
```
